Mata Nui Online Game II: The Final Chronicle – README

Debug Keys

To activate the in-game debug keys type kapura (must be the first thing typed unless the Insert key is pressed to reset). This will reveal the FPS counter and enable several debug keys.

Key Action
0 Toggle debug output visibility
1 Hide grid
2 Show grid
8 Reload scene
+ Mute sound
* Unmute sound
